
# 🤖 Multi-Agent AI System

This project is a Python-based Multi-Agent AI system where multiple intelligent agents work together to solve a user query step-by-step.

---

## 🧠 Project Overview

The system follows a structured AI workflow where each agent has a specific role:

### 🔍 Research Agent
- Fetches information using a web search API (DuckDuckGo)
- Collects raw data based on user query

### 🧠 Summarizer Agent
- Processes raw information
- Converts it into a short and simple explanation

### ✍️ Formatter Agent
- Formats the final output in a structured and readable format

---

## 🔁 Workflow

User Input  
→ Research Agent  
→ Summarizer Agent  
→ Formatter Agent  
→ Final Answer

---

## 🛠️ Technologies Used
